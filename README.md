# network-diagnostic-tool

## Overview

A menu-driven Bash scripting tool designed for network troubleshooting and diagnostics.

## Features

- Ping Test
- DNS Lookup
- Traceroute
- Port Connectivity Check
- Logging
- Report Generation

## Technologies Used

- Bash Scripting
- Linux Networking
- Ping
- nslookup
- traceroute
- netcat

## How to Run

chmod +x network_tool.sh

./network_tool.sh

## Menu

1. Ping Host
2. DNS Lookup
3. Trace Route
4. Port Check
5. Exit

## Skills Demonstrated

- Network Troubleshooting
- TCP/IP
- Linux Networking
- Shell Scripting
